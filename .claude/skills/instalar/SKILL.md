# /instalar — Configuración inicial de ValiOS

Se ejecuta una única vez para configurar ValiOS para el negocio del cliente.

## Verificación inicial

Antes de empezar, verifica:
- Si la carpeta aún se llama "ValiOS-ES" → sugerir renombrarla con el nombre del negocio
- Si `_memoria/empresa.md` ya tiene contenido real → preguntar si quiere reconfigurar

## Perfil del cliente

Primero, identifica el tipo de negocio:
> "Este ValiOS es para: (1) Profesional autónomo, (2) Freelancer o consultor, (3) Pequeña empresa, (4) Agencia o equipo?"

## Entrevista — una pregunta a la vez

1. "¿Cuál es el nombre de tu negocio y qué haces en pocas palabras?"
2. "¿Quién es tu cliente ideal — a quién sirves y qué problema resuelve?"
3. "¿Cuáles son tus principales servicios o productos?"
4. "¿Cómo quieres que escriba? (formal, cercano, técnico, informal?) — dame un ejemplo real si puedes."
5. "¿Hay palabras, frases o temas que nunca deba usar?"
6. "¿En qué idioma trabajas principalmente?"
7. "¿Cuál es tu mayor desafío o prioridad ahora?"
8. "¿Ya tienes web o redes sociales activas? ¿Cuál es el enlace?"
9. "¿Tienes logo, paleta de colores o referencias visuales? Descríbelas o pégalas aquí."
10. "¿Hay alguna herramienta que uses regularmente? (ej: Canva, Google Ads, Notion...)"

## Rellenar los archivos de memoria

Con las respuestas, rellena completamente:
- `_memoria/empresa.md` — nombre, descripción, clientes, servicios, herramientas
- `_memoria/preferencias.md` — tono, estilo, idioma, qué evitar
- `_memoria/estrategia.md` — foco actual, prioridades, web y redes activas
- `identidad/design-guide.md` — logo, colores, fuentes, referencias visuales

No dejes campos en blanco. Si la respuesta fue incompleta, registra lo que se dijo y márcalo para completar después.

## Resumen final

Muestra un resumen de lo configurado en máximo 5 líneas.

Sugiere renombrar la carpeta si aún tiene el nombre genérico.

Termina con:
> "ValiOS configurado para [nombre del negocio]. Usa /abrir para comenzar cualquier sesión de trabajo."
