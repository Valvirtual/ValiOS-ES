# /nuevo-proyecto — Crear carpeta de proyecto

Crea una estructura de carpeta organizada para un nuevo proyecto dentro del negocio.

## Qué hacer

1. Pregunta el nombre del proyecto si no fue dado
2. Crea la carpeta en `salidas/[nombre-del-proyecto]/`
3. Dentro, crea subcarpetas según el tipo de proyecto

## Estructura estándar

```
salidas/nombre-del-proyecto/
  briefing.md       ← objetivo, público, plazo
  contenido/        ← textos producidos
  visuales/         ← imágenes y artes
  entregables/      ← versión final para el cliente
```

## Rellenar el briefing

Pregunta:
- "¿Cuál es el objetivo de este proyecto?"
- "¿Para quién es? (cliente, plataforma, canal)"
- "¿Cuál es el plazo?"

Rellena `briefing.md` con las respuestas.

## Confirmar

> "Proyecto [nombre] creado en salidas/[nombre]. Ya puedes empezar."
