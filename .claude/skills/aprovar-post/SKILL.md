# /aprobar-post — Publicar contenido en múltiples canales

Prepara y formatea un post para publicación simultánea en blog, Instagram y Facebook.

## Antes de empezar

Lee `_memoria/preferencias.md` y `identidad/design-guide.md`.

## Recibir el contenido

Pide el texto del post o el archivo a publicar. Puede venir de `marketing/` si ya fue producido.

## Adaptar para cada canal

### Blog / Web
- Verificar título con palabra clave
- Añadir meta description (hasta 155 caracteres)
- Confirmar estructura: introducción, desarrollo, conclusión con CTA
- Formato: Markdown o HTML según el CMS del negocio

### Instagram
- Leyenda adaptada: gancho en las primeras 2 líneas (antes del "ver más")
- Emojis si el tono del negocio los usa
- Hashtags: 5-10 relevantes (mezcla de grandes y nicho)
- CTA claro al final

### Facebook
- Versión ligeramente más larga que Instagram
- Tono más conversacional
- Sin hashtags excesivos (máximo 3)
- Pregunta para generar comentarios si es adecuado

## Checklist antes de entregar

- [ ] Tono consistente con el negocio
- [ ] Sin errores ortográficos
- [ ] CTA presente en todos los canales
- [ ] Hashtags relevantes y actuales
- [ ] Imagen sugerida (describir si no hay)

## Output

Entregar los 3 formatos listos para copiar, claramente separados e identificados.
